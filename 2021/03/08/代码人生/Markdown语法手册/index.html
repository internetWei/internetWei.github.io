    前言

    关于 Markdown 的文章，已经有很多文章了，这篇文章的目的并不是教大家如何入门；它更像一本语法手册，供你随时翻阅。

    Markdown虽然不难，但是有很多语法由于不常用，所以用的时候难免会忘记怎么写，建议把这篇文章读一遍，然后收藏起来(最好下载到本地)，需要用到的时候再打开看一下。

    Markdown有很多高级语法很多平台并不支持，如果遇到平台不支持文章中的某些语法属于正常情况。
